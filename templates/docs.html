API Documentation

GET /info

Returns server information including storage used, total uploads, and total users.

Response:

{
            "storage_used": "XX MB",
            "uploads": XX,
            "users": XX
            }

POST /upload

Uploads a file to the server.

Request Body:

File: (form-data)

Response:

{
            "file_url": "https://kuuichi.xyz/generated_name.ext",
            "file-size": "XX MB",
            "file-type": "image/png",
            "date-uploaded": "YYYY-MM-DD HH:MM:SS",
            "delete_url": "https://kuuichi.xyz/delete/UUID"
            }

GET /uploads/{username}/{file_name}

Serves the requested file.

DELETE /delete/{delete_uuid}

Deletes a file based on the provided UUID.

GET /files

Lists all files for the authenticated user.

Response:

{
            "files": [
            {
            "file_name": "example.txt",
            "file_url": "https://kuuichi.xyz/uploads/username/example.txt",
            "file-size": "XX KB",
            "file-type": "txt",
            "date-uploaded": "YYYY-MM-DD HH:MM:SS",
            "delete_url": "https://kuuichi.xyz/delete/UUID"
            },
            ...
            ]
            }

GET /search

Search for files based on a query string.

Query Parameters:

query: The search term to match file names.

Response:

{
            "results": [
            {
            "file_name": "example.txt",
            "file_url": "https://kuuichi.xyz/uploads/username/example.txt",
            "score": 90
            },
            ...
            ]
            }
